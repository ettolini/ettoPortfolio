Humble To-Do List

( HTML - CSS - JavaScript )

As the name implies, this is a To-Do List project. With it, you can perform simple actions you'd expect from any list application: you can input and add some text that represents your task to your list, you can click on any added task to mark them as "completed", and you can also remove any added task in case you don't want to see it in your list anymore.

Text Tower

( C# )

This project is an executable, it functions as a stack storage. But users don't need to understand what a stack is, so I've built it to make it look like a "tower made of text blocks". After you close the program, a new .csv file will be created, said file will contain the text tower you built during execution. Also, keep this a secret between us, but I've heard you can unlock a fourth action if you build a tall enough tower.
